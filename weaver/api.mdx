> Jurisdiction: Universal Translator Registry
> Client: `src/services/api.ts` (`apiService`, `nexusApi`)
> Proxy: `next.config.ts` rewrites `/api/v1/*` → Nexus `:8001`

## Base URLs

| Surface | URL |
| :------ | :-- |
| Browser → Nexus (proxied) | `/api/v1` |
| Direct Nexus (curl, OpenAPI) | `http://localhost:8001/api/v1` |
| OpenAPI | `http://localhost:8001/docs` |
| Appwrite (client SDK) | `NEXT_PUBLIC_APPWRITE_ENDPOINT` |
| Whole-app env | `/api/observatory` (Next route, not Nexus) |

Auth: `nexusApi` attaches `Authorization: Bearer <session>` from Appwrite via `getSovereignSigil()`.

Router mount list: `nexus/router_registry.py`.
