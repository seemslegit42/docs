<Info>
  **Prerequisites**
  - [Bun](https://bun.sh) 1.1+
  - This `docs/` directory with a valid `docs.json`
</Info>

## Preview the docs site

From this repository:

```bash
bun run dev
```

From the parent **Weaver IDE** repository:

```bash
bun run docs:dev
```

The preview is served at `http://localhost:3000` by default.

## Custom port

```bash
bunx mint dev --port 3333
```

## Update the CLI

```bash
bun run update
```

## Validate links

```bash
bun run broken-links
```

## Run the Weaver app (IDE repo)

Documentation preview is separate from the product stack. To start Weaver Studio and Nexus:

```bash
cd ..   # IDE repo root (submodule parent)
bun run dev
```

| Service | URL |
| :------ | :-- |
| Frontend | http://localhost:3000 |
| Nexus API | http://localhost:8001 |

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port already in use">
    Mintlify will try the next available port, or pass `--port` explicitly.
  </Accordion>
  <Accordion title="CLI out of date vs production">
    Run `bun run update` in this directory.
  </Accordion>
  <Accordion title="Reset Mintlify cache">
    Remove `~/.mintlify` and run `bun run dev` again.
  </Accordion>
</AccordionGroup>
