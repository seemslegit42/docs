## Prerequisites

- [Bun](https://bun.sh) and Node 20+
- Python 3.11+ with `uv` or `pip`
- Docker (Dragonfly and optional deps via `weave` / compose)

## Start the stack

From the **parent IDE repository** (not this docs repo):

```bash
bun dev
# or: ./weave
# or: make dev
```

| Service | URL |
| :------ | :-- |
| Frontend (Weaver Studio) | http://localhost:3000 |
| Nexus API | http://localhost:8001 |
| Readiness | http://localhost:8001/api/v1/ready |
| Env observatory | http://localhost:3000/observatory |

## Environment

One file: **`.env`** at repo root (not `nexus/.env`).

```bash
bun run env:check      # CLI summary
bun run env:unify      # normalize sections
```

Visual checklist: [Environment](/weaver/environment).

## Useful commands

```bash
make dev-backend    # API only
make dev-frontend   # Next.js only
make stop           # stop .backend.pid / .frontend.pid
make logs-backend
bun run fresh       # reset caches, local DB, deps
```
