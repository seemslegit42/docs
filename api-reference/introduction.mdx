| :------ | :-- |
| Nexus Swarm API | `http://localhost:8001/api/v1` |
| Readiness | `http://localhost:8001/api/v1/ready` |
| Weaver Studio | `http://localhost:3000` |

## Authentication

Weaver uses **Appwrite** session cookies on the frontend and server-side admin keys for privileged routes. See [Authentication](/weaver/authentication) for sigil sovereignty and SSR uplinks.
